4 Bootstrap Carousel

A carousel vezérlő gombok data-bs-target értékének meg kell egyeznie a carousel div id értékével! Ha kettő van, különböző id kell!

<!-- CAROUSEL ALAP STRUKTÚRA -->
<div id="carouselEgy" class="carousel slide" data-bs-ride="carousel">

  <div class="carousel-inner">
    <div class="carousel-item active">   <!-- az első kép ACTIVE legyen! -->
      <img src="img/kep1.jpg" class="d-block w-100" alt="kép">
      <!-- Opcionális felirat: -->
      <div class="carousel-caption d-none d-md-block hatter">
        <h4>Felirat</h4>
      </div>
    </div>
    <div class="carousel-item">
      <img src="img/kep2.jpg" class="d-block w-100" alt="kép">
    </div>
  </div>

  <!-- Gombok: data-bs-target="#carouselEgy" -- egyezzen az id-vel! -->
  <button class="carousel-control-prev" type="button"
          data-bs-target="#carouselEgy" data-bs-slide="prev">
    <span class="carousel-control-prev-icon" aria-hidden="true"></span>
    <span class="visually-hidden">Previous</span>
  </button>
  <button class="carousel-control-next" type="button"
          data-bs-target="#carouselEgy" data-bs-slide="next">
    <span class="carousel-control-next-icon" aria-hidden="true"></span>
    <span class="visually-hidden">Next</span>
  </button>

</div>

<!-- HA KÉT CAROUSEL VAN, KÜLÖNBÖZŐ ID KELL! -->
<div id="carouselKetto" class="carousel slide" data-bs-ride="carousel">
  ...
  <button data-bs-target="#carouselKetto" ...></button>
</div>

⚠️ Leggyakoribb hiba:

Ha két carousel van és ugyanazt az id-t kapják, a gombok csak az egyiket vezérlik. Adj különböző id-t mindkettőnek, és a gombokban is a megfelelő id-t használd!

8 XML fájl felépítése

Az XML fájl hierarchikus adattárolás. Ha a feladat txt fájlból kér XML-t, nézd meg hány sor = 1 rekord!

autok.txt (forrás)
Suzuki       ← marka
S-cross      ← modell
2022         ← evjarat
105KW        ← teljesitmeny
Ford         ← következő auto...
Tourneo Custom
2021
92KW
autok.xml (eredmény)
<?xml version="1.0" encoding="UTF-8"?>
<autok>
  <auto>
    <marka>Suzuki</marka>
    <modell>S-cross</modell>
    <evjarat>2022</evjarat>
    <teljesitmeny>105KW</teljesitmeny>
  </auto>
  <auto>
    <marka>Ford</marka>
    <modell>Tourneo Custom</modell>
    <evjarat>2021</evjarat>
    <teljesitmeny>92KW</teljesitmeny>
  </auto>
</autok>

📋 XML szabályok (amiket be kell tartani):

  • • Mindig legyen <?xml version="1.0" encoding="UTF-8"?> az első sorban
  • • Van egy gyökérelem (pl. <autok>) ami mindent keretez
  • • Minden nyitó tagnak legyen záró tagja: <marka>...</marka>
  • • A tagok neve ne tartalmazzon szóközt, ékezetes betűt, ne kezdődjön számmal

9 JavaScript + XML feldolgozás

⚠️ FONTOS: fetch() vs XMLHttpRequest vs beágyazott XML

Módszer file:// protokoll Szerver (localhost) Megjegyzés
fetch() ❌ BLOKKOLVA ✅ Működik CORS blokkolja file://-n
XMLHttpRequest ⚠️ Firefox igen, Chrome nem ✅ Működik Inkább Firefox-ban
Beágyazott XML string ✅ MINDIG OK ✅ Működik → VIZSGÁN EZT HASZNÁLD!
✅ AJÁNLOTT: kocsik.js — XML beágyazva (szerver nélkül is működik!)
// Az XML adatot közvetlenül a JS-ben tároljuk — MINDIG működik!
let xmlString = `<?xml version="1.0" encoding="UTF-8"?>
<autok>
  <auto>
    <marka>Suzuki</marka>
    <modell>S-cross</modell>
    <evjarat>2022</evjarat>
    <teljesitmeny>105KW</teljesitmeny>
  </auto>
  <auto>
    <marka>Ford</marka>
    <modell>Tourneo Custom</modell>
    <evjarat>2021</evjarat>
    <teljesitmeny>92KW</teljesitmeny>
  </auto>
</autok>`;

// XML string → XML dokumentum objektummá alakítás
let parser = new DOMParser();
let xmlDoc = parser.parseFromString(xmlString, "text/xml");

// SELECT feltöltése az XML adatokból
let autok = xmlDoc.getElementsByTagName("auto");   // összes <auto> elem
let select = document.querySelector("select");      // a HTML select elem

// Üres alapelem hozzáadása
let alapOption = document.createElement("option");
alapOption.text = "-- Válassz --";
alapOption.value = "";
select.add(alapOption);

// Minden <auto> elemhez option létrehozása
for (let i = 0; i < autok.length; i++) {
    let option = document.createElement("option");
    // Tagból szöveg kinyerése:
    option.text = autok[i].getElementsByTagName("marka")[0].textContent;
    option.value = i;   // az index lesz az érték
    select.add(option);
}

// A function neve a feladat szerint: marka(parameter)
function marka(parameter) {
    let index = parameter.value;   // a kiválasztott option value-ja (index)

    // Ha az üres opciót választják:
    if (index === "") {
        document.getElementById("modell").value = "";
        document.getElementById("evjarat").value = "";
        document.getElementById("teljesitmeny").value = "";
        return;
    }

    // A kiválasztott <auto> elem lekérése index alapján
    let kivalasztott = xmlDoc.getElementsByTagName("auto")[index];

    // Inputok feltöltése az XML adatokból
    document.getElementById("modell").value =
        kivalasztott.getElementsByTagName("modell")[0].textContent;
    document.getElementById("evjarat").value =
        kivalasztott.getElementsByTagName("evjarat")[0].textContent;
    document.getElementById("teljesitmeny").value =
        kivalasztott.getElementsByTagName("teljesitmeny")[0].textContent;
}
Alternatíva: XMLHttpRequest (Firefox-ban vagy szerverrel működik)
let xhr = new XMLHttpRequest();
xhr.open("GET", "autok.xml", false);   // false = szinkron betöltés
xhr.send();

let parser = new DOMParser();

// ... többi kód ugyanaz

program.js — teljes megoldás magyarázattal
// fetch() betölti a JSON fájlt (csak szerverről működik!)
fetch("adatok.json")
.then(valasz => valasz.json())   // szöveg → JS objektum
.then(adatok => {                // adatok = a teljes JSON objektum

    const select = document.getElementById("varosLista");
    const div = document.getElementById("idojaras");

    // SELECT feltöltése: minden városhoz option
    adatok.varosok.forEach(v => {          // forEach = végigmegy a tömbön
        let option = document.createElement("option");
        option.textContent = v.nev;        // a városi neve lesz a szöveg ÉS az érték
        select.appendChild(option);        // hozzáadás a selecthez
    });

    // Megjelenít függvény: városnév alapján kikeresi és kiírja az adatokat
    function megjelenit(varosNev) {
        // find() megkeresi az első egyező elemet a tömbben
        let v = adatok.varosok.find(x => x.nev === varosNev);

        // innerHTML-be HTML kódot írunk (mértékegységekkel együtt!)
        div.innerHTML =
            "<h3>" + v.nev + "</h3>" +
            "Hőmérséklet: " + v.homerseklet + " °C<br>" +
            "Szél: " + v.szel + " km/h<br>" +
            "Időjárás: " + v.allapot;
    }

    // Első város automatikus megjelenítése (0. index = első elem)
    megjelenit(adatok.varosok[0].nev);

    // Eseményfigyelő: ha változik a select, frissíti az adatokat
    select.addEventListener("change", function() {
        megjelenit(this.value);   // this.value = a kiválasztott option szövege
    });

});

JSON vs XML — mikor melyik
JSON (adatok.json)
✅ Könnyű olvasni/írni
✅ JS natívan kezeli (.json())
✅ Modern feladatoknál tipikus
⚠️  fetch() kell → csak szerverről

XML (autok.xml)
✅ Hagyományos, régi feladatoknál
✅ Beágyazva JS-be = szerver nélkül
⚠️  Több kód kell (DOMParser)

JSON adatelérés:
adatok.varosok[0].nev    → "Budapest"
adatok.varosok[0].szel   → 15

XML adatelérés:
xmlDoc.getElementsByTagName("auto")[0]
  .getElementsByTagName("marka")[0]
  .textContent               → "Suzuki"

forEach vs for ciklus

// forEach (modern):
adatok.varosok.forEach(v => {
    console.log(v.nev);
});

// for (klasszikus):
for (let i = 0; i < adatok.varosok.length; i++) {
    console.log(adatok.varosok[i].nev);
}

find() — keresés tömbben

// Megkeresi az első olyat,
// ahol x.nev === "Budapest"
let v = adatok.varosok.find(
    x => x.nev === varosNev
);
// v.homerseklet → 22
// v.allapot     → "napos"

🔗 Anchor link a section-ra

A Bootstrap JS-t előbb töltsd be, mint a saját JS-t. Mindkettő legyen a </body> előtt. Ha a JS módosít HTML elemeket, az elemeknek már léteznie kell mikor a JS fut.

📐 col-* összesítés = 12

Egy sorban lévő col-*-ok összege 12 kell legyen azonos breakpointon. Pl. 4 oszlop = col-md-3 (4×3=12). Ha nem összegzik 12-re, törés vagy üres hely lesz.

🏷️ favicon típus

PNG ikonhoz: type="image/png". ICO fájlhoz: type="image/x-icon". Ha typo van (pl. imgage/png), a favicon nem jelenik meg!

⚙️ js/app.js — teljes JavaScript logika 
document.addEventListener('DOMContentLoaded', function () {

    // ⚠️ KULCS: minden adatfüggő kód a .then()-en BELÜL kell!
    // Ha kívül van: az adatok még nem töltődtek be (async!).
    fetch('adatok.json')
        .then(function (response) {
            return response.json();       // szöveg → JS objektum
        })
        .then(function (data) {
            const telefonok = data.telefonok;  // a JSON "telefonok" tömbje
            const select = document.getElementById('markaSelect');

            // SELECT feltöltése: minden telefonhoz egy <option>
            telefonok.forEach(function (telefon) {
                const option = document.createElement('option');
                option.textContent = telefon.marka;   // látható szöveg
                option.value = telefon.marka;         // elküldött érték
                select.appendChild(option);
            });

            // CHANGE esemény: select DOM elemre kell (nem a tömbre!)
            select.addEventListener('change', function () {
                // find() = megkeresi az első egyező elemet
                const kivalasztott = telefonok.find(function (t) {
                    return t.marka === select.value;
                });

                const doboz = document.getElementById('telefonAdatok');
                if (kivalasztott) {
                    // Az adatokat az előre elkészített elemekbe írjuk
                    document.getElementById('marka').textContent = kivalasztott.marka;
                    document.getElementById('modell').textContent = 'Modell: ' + kivalasztott.modell;
                    document.getElementById('tarhely').textContent = 'Tárhely: ' + kivalasztott.tarhely;
                    document.getElementById('ar').textContent = 'Ár: ' + kivalasztott.ar + ' Ft';
                    doboz.classList.remove('d-none');  // elrejtés megszüntetése
                } else {
                    doboz.classList.add('d-none');     // üres választásnál elrejtés
                }
            });
        });

    // MODAL ŰRLAP eseménykezelő (nem fetch-függő, kívül is lehet)
    const form = document.getElementById('uzenetForm');
    if (form) {
        form.addEventListener('submit', function (e) {
            e.preventDefault();   // megakadályozza a lap újratöltését!

            const adatok = {
                nev:    document.getElementById('nev').value,
                email:  document.getElementById('email').value,
                uzenet: document.getElementById('uzenet').value
            };
            console.log('Elküldött adatok:', adatok);

            // Modal bezárása JS-ből (getInstance = már meglévő példány)
            const modal = bootstrap.Modal.getInstance(
                document.getElementById('kapcsolatModal')
            );
            if (modal) modal.hide();

            form.reset();  // mezők kiürítése
        });
    }
});

  • • dat.addEventListener: Tömbre nem lehet eseményfigyelőt rakni. Megoldás: a select DOM elemre kell.
